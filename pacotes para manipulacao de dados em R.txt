Falando sobre manipular os dados em R, depende muito de como os dados são fornecidos. Alguns chegam mais organizados e limpos e outros mais desorganizados.

Entretanto algumas atividades provavelmente irão fazer parte do processo.
E para isso usamos os pacotes dplyr e tidyr.




* tidyr

Este pacote é usado para Remodelagem de Dados, que é a tarefa de converter uma linha em coluna, uma coluna em linha, fazer algum tipo de formato nos dados.

gather() -> tudo que é uma linha vira coluna
spread() -> tudo que é uma coluna vira linha

separate() -> separa 1 coluna em 3 (ex: uma coluna de dada com dia, mês e ano onde podemos dividir em 3 				    colunas: coluna dia, coluna mês, coluna ano)

unite() -> junto colunas e coluna em 1 coluna


Podemos com apenas uma função acima mudar completamente o formato (shape) dos nossos dados.





* dplyr

Este pacote é usado para Transformação dos Dados, ele trabalha / complementa o que foi feito com o tidyr.


select() -> seleciona parte dos dados
filter() -> filtra os dados

group_by()  -> agrupar os dados
summarise() -> sumarizar os dados

- arrange() -> reorganizar as linhas em uma coluna ou mais de uma coluna

- mutate()  ->  realizar cálculos dentro de uma mesma coluna

- join() - unir dois ou mais datasets.
